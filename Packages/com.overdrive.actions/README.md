# Overdrive QuickActions

**Quick-Access Actions and Favorites for Unity**

Overdrive QuickActions is a Unity editor tool that provides instant access to Unity's menu commands and shortcuts through convenient popup interfaces. Speed up your workflow by quickly finding and executing any Unity action without navigating through menus.

## Features

- **Action Palette**: Search and execute any Unity menu item or shortcut
- **Favorites System**: Save frequently used actions for instant access
- **Fuzzy Search**: Find actions quickly with smart text matching
- **Keyboard Navigation**: Full keyboard control for efficient workflow
- **Mouse Position Popups**: Actions appear right where you need them

## Getting Started

### Action Palette

The Action Palette gives you access to all Unity menu items and shortcuts in one searchable interface.

**To open the Action Palette:**
- Press `A` key in the Scene View (customizable shortcut)

**Using the Action Palette:**
1. Type to search for any Unity action (e.g., "duplicate", "create cube", "save scene")
2. Use ↑/↓ arrow keys to navigate results
3. Press `Enter` to execute the selected action
4. Press `Escape` to close the palette
5. Click the ⭐ button next to any action to add it to your favorites

### Favorites

The Favorites popup shows your most-used actions for instant access.

**To open Favorites:**
- Hold `Alt` and right-click in the Scene View (customizable shortcut)

**Managing Favorites:**
- Add actions to favorites using the ⭐ button in the Action Palette
- Remove favorites by clicking the ❌ button in the Favorites popup
- Execute favorites by selecting them and pressing `Enter` or clicking

## Tips & Tricks

### Quick Search
- Type partial words: "dup" finds "Duplicate"
- Use spaces: "create light" finds "Create Light"
- Search by category: "gameobject" shows all GameObject actions

### Keyboard Workflow
1. Press `A` to open Action Palette
2. Start typing immediately (no need to click the search field)
3. Use ↑/↓ to navigate, `Enter` to execute
4. Build muscle memory for your favorite shortcuts

### Organizing Favorites
- Add your most-used actions to favorites for one-click access
- Group related actions (e.g., all lighting-related commands)
- Regularly review and clean up your favorites list

## Keyboard Shortcuts

| Shortcut | Action |
|----------|---------|
| `A` | Open/Close Action Palette |
| `Alt + Right Click` | Open/Close Favorites (Scene View only) |
| `↑/↓ Arrow Keys` | Navigate action list |
| `Enter` | Execute selected action |
| `Escape` | Close popup |

**Note:** These shortcuts can be customized in Unity's Shortcut Manager (**Edit → Shortcuts**). Search for "Overdrive" to find and modify the QuickActions shortcuts.
